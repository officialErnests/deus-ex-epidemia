1. step - pick a mask, each mask comes with adifrent flavour
<img width="347" height="193" alt="fQG0Wm" src="https://github.com/user-attachments/assets/09d125cc-9a98-4f33-9fb1-23f2e6d8ef26" /><br>

5. win - each time you win you unlock a new mask that can help you with game
